# Netflix Trends App

Interactive Streamlit dashboard for Netflix titles with:

- Robust cleaning (dates, countries, genres, duration, Originals heuristic)
- Visualizations (year trends, date-added, type split, ratings, durations)
- Geography (top countries, choropleth)
- Genres (co-occurrence heatmap)
- Reviews (HTML scraping **or** TMDB API)
